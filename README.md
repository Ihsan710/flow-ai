## 🛠️ System Deployment

### Prerequisites
- Node.js (v18+)
- Google Gemini API Key

### Initialization

1. **Clone the matrix:**
   ```bash
   git clone https://github.com/Ihsan710/flow-ai.git
   cd flow-ai
   ```

2. **Backend Setup:**
   ```bash
   cd backend
   npm install
   # Create a .env file and add your GEMINI_API_KEY
   node server.js
   ```

3. **Frontend Setup:**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
